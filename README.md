# Internship Task 2 – Strict Tech Stack Implementation

## Tech Stack Used
- Templating Engine: Nunjucks
- Bundler: Vite

## Folder Structure
- src/templates → Nunjucks templates
- src/assets → CSS and JavaScript files
- dist → Compiled HTML output

## How to Run
1. npm install
2. npm run dev
3. npm run build

## Description
This project recreates the Internship Task Document using Nunjucks templates and Vite bundler, ensuring the final output matches Task 1 exactly.
